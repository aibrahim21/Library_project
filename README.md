# Library Management System

A simple, interactive web application for managing a personal book collection. Built with vanilla JavaScript, HTML, and CSS to demonstrate object-oriented programming, DOM manipulation, and CRUD (Create, Read, Update, Delete) operations.

## Features
- **Add Books:** Input book name, price, and author details (name & email) through a validated form.
- **Display Books:** View all books in a dynamic, sortable table.
- **Edit Books:** Inline editing for any book entry with Save/Cancel options.
- **Delete Books:** Remove books from the collection with a single click.
- **Data Validation:** Ensures all fields are correctly filled before adding or saving.

## Technologies Used
- **JavaScript (ES6):** Object constructors, array methods, event handling.
- **HTML5:** Semantic structure, form elements.
- **CSS3:** Flexbox/Grid for responsive layout, custom styling.

## How to Use
1.  Enter the number of books you want to add.
2.  Fill in the details for each book in the generated form.
3.  Click "Add Book" to add it to your collection.
4.  Use the Edit/Delete buttons to manage your list.
